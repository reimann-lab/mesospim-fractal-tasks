# Installation

You can install the package either directly into your Fractal Python environment,
or inside a Fractal Conda environment used for task execution.

## Requirements

The package requires 3.11 or 3.12.

### Install from PyPI

```bash
pip install mesospim-fractal-tasks
```

### Install from source

```bash

git clone https://github.com/reimann-lab/mesospim-fractal-tasks.git
cd mesospim-fractaltasks
pip install -e .
```
